## 关键技术决策

### 为什么不用 pyobjc？
macOS AXAPI 通过 JXA (osascript -l JavaScript) 实现，零外部依赖。每个 macOS 都自带。

### 为什么不用 WebContainers？
Bolt.new 用 WebContainers 实现浏览器内 Node.js 运行，但依赖 URL 需联网下载 WASM 运行时且商业许可受限。MadCop 专注本地优先。

### 为什么 Plan 用同步 LLM 调用？
`client.chat()` 是同步方法（非 streaming），用于 plan 生成和验证异步 SSE 流里 `client.chat()` 在 Phase 0 调用，Phase 1 走正常 streaming。

### ModelSelector 为什么不直接写 provider 状态？
ModelSelector 通过 `@update:selected-model` 向上 emit，ChatInput 存 `selectedModel` ref，发消息时传给 `sendMessage(options.model)`。Provider 切换则通过 settings API 里的 `make_active` 实现。
